Road to Haxe 5.0

Dear Community,

We are currently working towards Haxe 5.0 which, as a major version, will include a number of breaking change.

We are maintaining a list of Haxe 5.0 breaking changes as we introduce them in development branch. You can also check the Haxe 5.0 Roadmap to get an overview of what we intend to include in the release.

To ease the migration from Haxe 4, we introduced -D haxe-next in Haxe 4.3. This define activates a number of warnings about things that will not work anymore in Haxe 5.0.

Change Log

A changelog draft is available on Haxe repository wiki and is being updated as we add major changes to development branch.
